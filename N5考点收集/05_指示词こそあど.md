# 指示詞（こそあど）

## こそあど体系总览

日语指示词按**距离感**分四组，每组对应不同的词性：

```
こ（近）─ 说话人身边
そ（中）─ 听话人身边
あ（远）─ 双方之外
ど（疑）─ 疑问
```

| 组 | 距离         | 核心含义           |
| -- | ------------ | ------------------ |
| こ | 说话人范围   | 这 / 我这里        |
| そ | 听话人范围   | 那 / 你那里        |
| あ | 双方之外     | 那 / 远处          |
| ど | 疑问         | 哪 / 什么          |

---

## 四类词性对照

指示词按后面接不接名词、表示什么类型，分为四类：

| 词性         | こ             | そ             | あ             | ど             |
| ------------ | -------------- | -------------- | -------------- | -------------- |
| 物（代词）   | **これ**       | **それ**       | **あれ**       | **どれ**       |
| 修饰名词     | **この** + N   | **その** + N   | **あの** + N   | **どの** + N   |
| 场所         | **ここ**       | **そこ**       | **あそこ**     | **どこ**       |
| 方向/礼貌   | **こちら**     | **そちら**     | **あちら**     | **どちら**     |

```
これ・それ・あれ = 单独用，后面不接名词
この・その・あの = 必须接名词，不能单独用
ここ・そこ・あそこ = 场所
こちら・そちら・あちら = 方向 or 礼貌替代
```

---

## 物：これ・それ・あれ・どれ

独立代词，指代事物。不接名词。

| 词     | 含义         | 使用场景                     |
| ------ | ------------ | ---------------------------- |
| これ   | 这个         | 在说话人手里/身边            |
| それ   | 那个         | 在听话人手里/身边            |
| あれ   | 那个（远处） | 两人手都够不着               |
| どれ   | 哪个         | 疑问（三选一或多选一）       |

### 距离示意

```
[说话人] ←これ→ [听话人] ←それ→ ←あれ→
  こ圈             そ圈            あ圈
```

### 例句

| 例句                         | 读法                           | 翻译               |
| ---------------------------- | ------------------------------ | ------------------ |
| これは 本です                | これは ほんです               | 这是书             |
| それは 何ですか              | それは なんですか             | 那是什么           |
| あれは 学校です              | あれは がっこうです           | 那是学校（远处）   |
| どれが あなたの ですか      | どれが あなたの ですか       | 哪个是你的         |

---

## 修饰名词：この・その・あの・どの

不能单独使用，后面必须接名词。

| 词       | 含义       | 用法           |
| -------- | ---------- | -------------- |
| この + N | 这个 N     | 说话人范围的 N  |
| その + N | 那个 N     | 听话人范围的 N  |
| あの + N | 那个 N     | 远处/双方共知的 N |
| どの + N | 哪个 N     | 疑问           |

### 例句

| 例句                         | 读法                           | 翻译               |
| ---------------------------- | ------------------------------ | ------------------ |
| この本は おもしろいです     | このほんは おもしろいです     | 这本书很有趣       |
| そのかばんは いくらですか   | そのかばんは いくらですか     | 那个包多少钱       |
| あの人を 知っていますか     | あのひとを しっていますか     | 你认识那个人吗     |
| どの電車に 乗りますか       | のでんしゃに のりますか       | 坐哪趟电车         |

---

## これ vs この（必考选择题）

这是 N5 最常见的选词题陷阱：

```
これ = 独立代词，单独用
この = 连体词，后面必须有名词
```

| ✓                           | ✗                           |
| --------------------------- | --------------------------- |
| これ**は** 本です           | これ 本です（缺は）         |
| この**本**は おもしろいです | この は 本です（缺名词）    |

### 考题对比

```
Q: （＿＿）は わたしの かさです。
   ① これ     ② この     ③ ここ     ④ こちら

后面有「は」没有名词 → 需要独立代词 → ① ✓
② この后面必须接名词，不能直接加は
```

```
Q: （＿＿）かばんは 田中さんのです。
   ① これ     ② この     ③ どれ     ④ あれ

后面有名词「かばん」→ 需要修饰名词的指示词 → ② ✓
①③④ 都是独立代词，不能直接加名词
```

## 场所：ここ・そこ・あそこ・どこ

| 词       | 含义         | 使用场景               |
| -------- | ------------ | ---------------------- |
| ここ     | 这里         | 说话人所在场所         |
| そこ     | 那里         | 听话人所在场所         |
| あそこ   | 那里（远处） | 两人都不在的场所       |
| どこ     | 哪里         | 疑问                   |

### 例句

| 例句                               | 读法                               | 翻译                 |
| ---------------------------------- | ---------------------------------- | -------------------- |
| ここは 図書館です                 | ここは としょかんです             | 这里是图书馆         |
| そこに 何が ありますか           | そこに なにが ありますか         | 那里有什么           |
| トイレは あそこです               | トイレは あそこです               | 厕所在那边           |
| 田中さんは どこですか             | たなかさんは どこですか           | 田中先生在哪         |

### 注意

- あそこ = **あそこ**，不是 あこ
- 「そこ」也可以指代对方刚才说的话（"你说的那个"）

---

## 方向/礼貌：こちら・そちら・あちら・どちら

| 词       | 含义           | 用法                                       |
| -------- | -------------- | ------------------------------------------ |
| こちら   | 这边 / 这位    | ここの礼貌版；介绍人物时也用               |
| そちら   | 那边 / 那位    | そこの礼貌版；电话中问对方                 |
| あちら   | 那边（远处）   | あそこの礼貌版                             |
| どちら   | 哪边 / 哪位    | どこの礼貌版；二选一时用                   |

### 例句

| 例句                               | 读法                               | 翻译                   |
| ---------------------------------- | ---------------------------------- | ---------------------- |
| こちらは 田中さんです             | こちらは たなかさんです           | 这位是田中先生         |
| 出口は あちらです                 | でぐちは あちらです               | 出口在那边             |
| お国は どちらですか               | おくには どちらですか             | 您来自哪个国家         |
| そちらは どなたですか（电话）     | そちらは どなたですか             | 您是哪位               |

> **こちら/そちら/あちら/どちら** 是 **ここ/そこ/あそこ/どこ** 的礼貌形式，对人/方向/场所均可使用。

---

## こそあど 用法汇总

| 词性       | こ（近） | そ（中） | あ（远） | ど（疑） | 接什么     |
| ---------- | -------- | -------- | -------- | -------- | ---------- |
| 物         | これ     | それ     | あれ     | どれ     | 单独用     |
| 连体（修饰） | この     | その     | あの     | どの     | + 名词     |
| 场所       | ここ     | そこ     | あそこ   | どこ     | 单独用     |
| 方向/礼貌  | こちら   | そちら   | あちら   | どちら   | 单独用     |

## 经典考题陷阱

### 1. これ vs この

```
Q: （＿＿）は 日本語の 辞書です。
   ① この     ② これ     ③ ここ     ④ どの

后面有「は」无名词 → 需要独立代词（物）→ ② ✓
① 后面必须接名词，这里没有
③ 是场所（这里是日语词典？不自然）
④ 是疑问修饰，后面缺名词
```

### 2. 对话中的 そ 切换

```
A: それは 何ですか。
B: これは ペンです。

A 看 B 手里的东西，用 それ（在对方那儿）
B 回答自己手里的东西，用 これ（在自己这儿）

同一个东西，A 用 そ，B 用 こ —— 视角切换是高频考点。
```

### 3. あそこ vs あちら

```
Q: すみません、駅は（＿＿）ですか。
   ① どこ     ② あそこ     ③ あちら     ④ どちら

问路人 → 不确定远近，用 どこ 或 あちら 都可
但 ③ あちら 比 ② あそこ 更礼貌
考试中两句都说得通时 → 选更礼貌的那一个
如果是店内问店员："トイレは（＿＿）ですか" → あちら 优于 あそこ
```

### 4. どれ vs どの — 选择

```
Q: （＿＿）が あなたの かばんですか。（三选一/多选一）
   ① どれ     ② どの     ③ どこ     ④ どちら

疑问词作主语 + 三选一 → どれ → ① ✓
② 後面缺名词
③ 是场所
④ 是二选一
```

### 5. 電話中的 そちら

```
电话场景：双方不在同一地点，不知道对方是谁
→ そちらは どなたですか（您是哪位？）
→ 用 そちら 指代电话那头，固定用法
```

## 速记口诀

```
これこのこここちら — こ 是"我这边"
それそのそこそちら — そ 是"你那边"
あれあのあそこあちら — あ 是"咱们都够不着"
どれどのどこどちら — ど 是"哪一个是"
これ单独この接名词 — 有 N 无 N 是关键
对话切换视角看 — 你そ我こ 别搞反
```
