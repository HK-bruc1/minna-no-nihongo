# 动词分类与变形

## 动词三类

日语动词按变形规律分三类：

| 类别       | 日文名         | 特征                                   | 例子               |
| ---------- | -------------- | -------------------------------------- | ------------------ |
| 五段動詞   | ごだんどうし   | 词尾在五段（あいうえお）之间变化       | 書**く**、読**む** |
| 一段動詞   | いちだんどうし | 词尾固定，后接る，变形简单             | 食**べる**、見**る** |
| 変格動詞   | へんかくどうし | 完全不规则，只有两个                   | **する**、**来る** |

```
五段 = 规则1（变化多，五个段跳来跳去）
一段 = 规则2（简单，只在一个段上）
変格 = 不规则（就两个，死记）
```

---

## 五段 vs 一段的区分方法

这是 N5 最高频的"分类判断题"。正确区分才能正确变形。

### 判断规则

```
动词基本形以「る」结尾，且「る」前面的假名在 い段 或 え段 → 一段动词
其他所有情况 → 五段动词
```

### 例

| 动词     | る前假名   | い段/え段？       | 判断     |
| -------- | ---------- | ----------------- | -------- |
| 食べる   | べ（え段） | ✅ え段           | **一段** |
| 見る     | み（い段） | ✅ い段           | **一段** |
| 起きる   | き（い段） | ✅ い段           | **一段** |
| 書く     | く         | ❌ 不在い/え段    | **五段** |
| 読む     | む         | ❌ 不在い/え段    | **五段** |
| 買う     | う         | ❌ 不在い/え段    | **五段** |
| 話す     | す         | ❌ 不在い/え段    | **五段** |

### 陷阱：看起来像一段、实际是五段的例外

有些动词以「る」结尾且る前是い段/え段，但它们是**五段动词**：

| 动词     | 看似一段？     | 实际   | ます形       |
| -------- | -------------- | ------ | ------------ |
| 入る     | は**い**る     | 五段   | 入**り**ます |
| 帰る     | か**え**る     | 五段   | 帰**り**ます |
| 切る     | **き**る       | 五段   | 切**り**ます |
| 知る     | **し**る       | 五段   | 知**り**ます |
| 走る     | はし**る**     | 五段   | 走**り**ます |
| 要る     | **い**る       | 五段   | 要**り**ます |

> N5 必考一组：**入る/帰る/切る/知る**。这四个全部是五段动词，ます形中会露出「り」。

### 速记

> **る前い段/え段 = 大概率一段，但要背例外清单（入る・帰る・切る・知る・走る・要る）**

| 变形       | 用途                     | 考频     |
| ---------- | ------------------------ | -------- |
| ます形     | 礼貌体、其他变形的基础   | ⭐⭐⭐⭐⭐ |
| て形       | 连接/请求/进行/试着~     | ⭐⭐⭐⭐⭐ |
| た形       | 过去/完成/経験           | ⭐⭐⭐⭐⭐ |
| ない形     | 否定/禁止/必须~          | ⭐⭐⭐⭐   |
| 辞書形     | 词典形态/简体/能力       | ⭐⭐⭐⭐   |
| 意志形     | 劝诱/打算/~と思う        | ⭐⭐⭐     |

## ます形

### 功能

礼貌体（丁寧体）。从辞书形变来，也可反推辞书形。

### 变化规则

| 类别       | 规则                             |
| ---------- | -------------------------------- |
| 五段       | 词尾う段 → **い段** + ます       |
| 一段       | 去る + ます                      |
| する       | します                           |
| 来る       | きます                           |

### 例

| 辞書形 | 类别   | ます形             |
| ------ | ------ | ------------------ |
| 書く   | 五段   | 書**き**ます       |
| 読む   | 五段   | 読**み**ます       |
| 話す   | 五段   | 話**し**ます       |
| 買う   | 五段   | 買**い**ます       |
| 待つ   | 五段   | 待**ち**ます       |
| 死ぬ   | 五段   | 死**に**ます       |
| 遊ぶ   | 五段   | 遊**び**ます       |
| 食べる | 一段   | 食べます           |
| 見る   | 一段   | 見ます             |
| する   | 変格   | します             |
| 来る   | 変格   | きます             |

### ます形反过来判断辞书形

```
書きます → 去ます → 書き → い段 → 对应う段 → 書く ✓
食べます → 去ます → 食べ → 一段（去る前是え段）→ 食べる ✓
```

## て形（必考重点）

N5 分值最高的变形。て形本身用于连接、请求、进行等句型，同时也是た形的基础（て→た 直接替换）。

### 一段动词

去る + て。

| 辞書形 | て形       |
| ------ | ---------- |
| 食べる | 食べ**て** |
| 見る   | 見**て**   |
| 起きる | 起き**て** |

### 変格动词

| 辞書形 | て形     |
| ------ | -------- |
| する   | **して** |
| 来る   | **きて** |

### 五段动词 — 音便（おんびん）规律

五段のて形是 N5 最难的部分，词尾会发生**音便**（发音变化）。根据词尾不同分四类：

```
        五段て形 音便分类

く・ぐ ──→ い音便
う・つ・る ──→ 促音便
ぬ・ぶ・む ──→ 撥音便
す ──→ して（无音便）
```

### い音便 — く・ぐ

| 辞書形 | 词尾 | て形           | 规则               |
| ------ | ---- | -------------- | ------------------ |
| 書く   | く   | 書**いて**     | く → い + て       |
| 聞く   | く   | 聞**いて**     | く → い + て       |
| 歩く   | く   | 歩**いて**     | く → い + て       |
| 泳ぐ   | ぐ   | 泳**いで**     | ぐ → い + **で**   |
| 急ぐ   | ぐ   | 急**いで**     | ぐ → い + **で**   |

> **ぐ 结尾时用 で（浊音），不读 て。** 这是高频陷阱。

### 促音便 — う・つ・る

| 辞書形 | 词尾 | て形           | 规则               |
| ------ | ---- | -------------- | ------------------ |
| 買う   | う   | 買**って**     | う → っ + て       |
| 会う   | う   | 会**って**     | う → っ + て       |
| 待つ   | つ   | 待**って**     | つ → っ + て       |
| 持つ   | つ   | 持**って**     | つ → っ + て       |
| 帰る   | る   | 帰**って**     | る → っ + て       |
| 作る   | る   | 作**って**     | る → っ + て       |
| ある   | る   | あ**って**     | る → っ + て       |

### 撥音便 — ぬ・ぶ・む

| 辞書形 | 词尾 | て形           | 规则               |
| ------ | ---- | -------------- | ------------------ |
| 死ぬ   | ぬ   | 死**んで**     | ぬ → ん + **で**   |
| 遊ぶ   | ぶ   | 遊**んで**     | ぶ → ん + **で**   |
| 呼ぶ   | ぶ   | 呼**んで**     | ぶ → ん + **で**   |
| 読む   | む   | 読**んで**     | む → ん + **で**   |
| 飲む   | む   | 飲**んで**     | む → ん + **で**   |

> **撥音便后用で（浊音），不用て。**

### 无音便 — す

| 辞書形 | 词尾 | て形           | 规则           |
| ------ | ---- | -------------- | -------------- |
| 話す   | す   | 話**して**     | す → し + て   |
| 出す   | す   | 出**して**     | す → し + て   |

> す 不触发音便，直接 す→し+て，和ます形一样的 し。

### 特殊例外

| 辞書形 | て形           | 说明                           |
| ------ | -------------- | ------------------------------ |
| 行く   | 行**って**     | 促音便！不是 い音便（✗ 行いて） |

> 唯一一个词尾是 く 却走促音便的动词。**行く→行って**，N5 必考。

### て形音便 速查总表

| 词尾         | 音便类型   | て/で       | 例                         |
| ------------ | ---------- | ----------- | -------------------------- |
| く           | い音便     | いて        | 書く→書いて               |
| ぐ           | い音便     | **いで**    | 泳ぐ→泳いで               |
| う・つ・る   | 促音便     | って        | 買う→買って 待つ→待って   |
| ぬ・ぶ・む   | 撥音便     | **んで**    | 死ぬ→死んで 遊ぶ→遊んで   |
| す           | 无音便     | して        | 話す→話して               |
| 行く（例外） | 促音便     | って        | 行く→行って               |

### 速记口诀

```
く→いて、ぐ→いで
うつる→って
ぬぶむ→んで
す→して
行く→行って 是例外
```

### い形容詞て形与动词て形的区分

N5 真题常见混淆点——两者都要变て形但规则不同：

| 类型       | 规则                 | 例                            |
| ---------- | -------------------- | ----------------------------- |
| 动词て形   | 按音便规则           | 書**いて**、泳**いで**        |
| い形容詞   | 去い + **くて**      | 高**くて**、安**くて**        |

```
泳いで（およいで）— 动词「泳ぐ」的て形、い音便
高くて（たかくて）— い形容词「高い」的て形、くて固定

二者结尾不同：泳いで vs 高くて
```

> 做题时先看原词是动词还是形容词，再套对应规则。不要因为"いで"和"くて"看起来像就混淆。

## た形

### 功能

过去、完了、経験（〜たことがある）。

### 变化规则

**た形 = て形，把 て/で 换成 た/だ。**

| て形       | → た形       |
| ---------- | ------------ |
| 書**いて** | 書**いた**   |
| 泳**いで** | 泳**いだ**   |
| 買**って** | 買**った**   |
| 死**んで** | 死**んだ**   |
| 話**して** | 話**した**   |

> て形掌握了，た形就自动掌握了。**两者一一对应**。

## ない形

### 功能

否定（〜ない）、禁止（〜ないでください）、必须（〜なければならない）。

### 变化规则

| 类别   | 规则                             |
| ------ | -------------------------------- |
| 五段   | 词尾う段 → **あ段** + ない       |
| 一段   | 去る + ない                      |
| する   | しない                           |
| 来る   | こない                           |

### 五段特别注意：う 结尾

| 辞書形 | 词尾 | あ段   | ない形           |
| ------ | ---- | ------ | ---------------- |
| 買う   | う   | **わ** | 買**わ**ない     |
| 会う   | う   | **わ** | 会**わ**ない     |

> **う → わ + ない**，不是「あ + ない」。う的あ段是わ。

### ある → ない

| 辞書形 | ない形     | 说明                         |
| ------ | ---------- | ---------------------------- |
| ある   | **ない**   | 不是 ✗ あらない，直接用ない |

### 例

| 辞書形 | 类别   | ない形             |
| ------ | ------ | ------------------ |
| 書く   | 五段   | 書**か**ない       |
| 読む   | 五段   | 読**ま**ない       |
| 話す   | 五段   | 話**さ**ない       |
| 買う   | 五段   | 買**わ**ない       |
| 待つ   | 五段   | 待**た**ない       |
| 死ぬ   | 五段   | 死**な**ない       |
| 遊ぶ   | 五段   | 遊**ば**ない       |
| ある   | 五段   | **ない**           |
| 食べる | 一段   | 食べ**ない**       |
| 見る   | 一段   | 見**ない**         |
| する   | 変格   | **しない**         |
| 来る   | 変格   | **こない**         |

## 辞書形（じしょけい）

### 功能

词典形态，也是简体现在时/将来时的肯定形。

- 辞書形 で いい（也可以）
- 辞書形 ことが できる（会/能做）
- 辞書形 まえに（在~之前）
- 辞書形 + の（名词化）

### 还原方法

从ます形反推辞書形：

```
ます形去掉ます → 得到い段/え段
→ 一段 or 変格：直接音即可
→ 五段：把い段推回对应的う段
```

| ます形     | 去ます | 判断             | → 辞書形     |
| ---------- | ------ | ---------------- | ------------ |
| 書きます   | 書き   | き=い段→推定五段 | 書**く**     |
| 食べます   | 食べ   | べ=え段→推定一段 | 食べ**る**   |
| します     | し     | 变格             | **する**     |
| きます     | き     | 变格             | **来る（くる）** |

## 意志形（いこうけい）

### 功能

劝诱、打算、意愿（〜と思う）。

### 变化规则

| 类别       | 规则                             |
| ---------- | -------------------------------- |
| 五段       | 词尾う段 → **お段** + う         |
| 一段       | 去る + よう                      |
| する       | しよう                           |
| 来る       | こよう                           |

### 例

| 辞書形 | 类别   | 意志形             |
| ------ | ------ | ------------------ |
| 書く   | 五段   | 書**こう**         |
| 読む   | 五段   | 読**もう**         |
| 話す   | 五段   | 話**そう**         |
| 買う   | 五段   | 買**おう**         |
| 待つ   | 五段   | 待**とう**         |
| 死ぬ   | 五段   | 死**のう**         |
| 遊ぶ   | 五段   | 遊**ぼう**         |
| 食べる | 一段   | 食べ**よう**       |
| 見る   | 一段   | 見**よう**         |
| する   | 変格   | **しよう**         |
| 来る   | 変格   | **こよう**         |

## 特殊变形：する动词（サ変）

「N+する」构成大量动词（勉強する、散歩する、電話する 等），全部按 する 的变形方式走。

| 变形   | する     | 勉強する           |
| ------ | -------- | ------------------ |
| ます形 | します   | 勉強します         |
| て形   | して     | 勉強して           |
| た形   | した     | 勉強した           |
| ない形 | しない   | 勉強しない         |
| 意志形 | しよう   | 勉強しよう         |

## 变形 → N5 句型联动

每种变形在 N5 考试中都对应固定句型。记住"变形是为了用"，做题时才不会选错。

| 变形   | N5 必考句型                                   | 例句                             |
| ------ | --------------------------------------------- | -------------------------------- |
| ます形 | 〜ます（礼貌体）                              | 毎日 勉強**します**              |
| ます形 | 〜ましょう（劝诱/一起~）                      | 一緒に 行**きましょう**          |
| ます形 | 〜たい（想~）[去ます+]                        | 日本へ 行**きたい**です          |
| ます形 | 〜方（〜的方法）[去ます+]                     | 作**り方**（つくりかた）         |
| ます形 | 〜に 行く（去~）[去ます+]                     | 食べ**に**行きます               |
| て形   | 〜て ください（请~）                          | 待**って** ください              |
| て形   | 〜て います（正在~/状态）                     | 雨が 降**って** います           |
| て形   | 〜て も いい です（可以~）                    | 写真を 撮**って** も いい です   |
| て形   | 〜て から（~之后~）                           | ご飯を 食べ**て** から           |
| た形   | 〜た こと が ある（有过~经历）                | 日本へ 行**った** こと が ある    |
| た形   | 〜た ほう が いい（最好~）                    | 早く 寝**た** ほう が いい       |
| ない形 | 〜ない で ください（请不要~）                 | タバコを 吸**わない** で ください |
| ない形 | 〜なければ ならない（必须~）                  | 勉強**しなければ** ならない       |
| 辞書形 | 〜こと が できる（能/会~）                    | 日本語を 話**す** こと が できる  |
| 辞書形 | 〜まえ に（在~之前）                          | 寝**る** まえ に                  |
| 意志形 | 〜と 思います（打算~）                        | 留学**しよう** と 思います        |
| 意志形 | 〜か どうか（是否~）                          | 行**こう** か どうか              |

> 核心逻辑：先判断句子需要什么句型，再确定动词变什么形。反过来推理也可以——看到变形就能读出用途。

## 经典考题陷阱

### 1. 五段て音便选择题

```
Q: 図書館で本を（＿＿）から、家に帰ります。
   ① かりて     ② かって     ③ かりって     ④ かいて

借りる（一段）→ 借りて → ① ✓
買う（五段・う→促音便）→ 買って → ②（这里是借りて的干扰项）
```

> 核心：先判断动词类别，再变形。混淆项通常用错误的音便规则制造。

### 2. 行く的て形

```
Q: デパートへ（＿＿）、買い物をしました。
   ① 行て       ② 行いて     ③ 行って       ④ 行で

行く→行って（促音便例外）→ ③ ✓
```

### 3. 入る/帰る 的类别判断

```
入る（はいる） — 虽然る前是い段 → 但它是五段动词
→ ます形：入ります（不是 ✗ 入ます）
→ て形：入って（不是 ✗ 入て）
```

### 4. ある → ない

```
ある 的否定不是 あらない，而是 ない。
```

### 5. 来る 全套干扰

```
来る → ます形：きます（✗ くます/こます）
来る → ない形：こない（✗ きない/くない）
来る → て形：きて（✗ くて/きって）
```

## 变形速记口诀

```

    くいて ぐいで うつる促 ぬぶむ撥 すして 行く例外
ない → ウ段跳到ア段 / う跳わ / ある→ない
```
