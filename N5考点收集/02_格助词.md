# 格助詞（かくじょし）

## 一览总表

| 助词 | 核心功能               | 经典例句                         |
| ---- | ---------------------- | -------------------------------- |
| は   | 主题提示               | 私は学生です                     |
| が   | 主语强调 / 对象        | 雨が降っています                 |
| を   | 动作对象 / 经过/离开   | ご飯を食べます                   |
| に   | 时间点/方向/对象/存在处 | 3時**に** / 学校**に**行く        |
| で   | 动作场所/手段/范围     | 図書館**で**勉強する             |
| へ   | 移动方向               | 東京**へ**行きます               |
| と   | 和~/引用               | 友達**と**遊ぶ                   |
| から | 起点/来源              | 9時**から**始まる                |
| まで | 终点                   | 5時**まで**終わる                |
| も   | 也                     | 私**も**学生です                 |
| の   | 所属/内容说明          | 私**の**本 / 日本語**の**勉強     |

以上助词中，**に** 和 **で** 各自承担多种功能，是 N5 考得最多、也最容易混淆的两个。

---

## は — 主题提示

### 功能

提示整个句子的**话题**（"关于这个，我说一下"），管到句末。

```
N + は
```

### 例句

| 例句                         | 读法                           | 翻译             |
| ---------------------------- | ------------------------------ | ---------------- |
| 私は田中です                 | わたしはたなかです             | 我是田中         |
| 今日は日曜日です             | きょうはにちようびです         | 今天是星期天     |
| 東京は人が多いです           | とうきょうはひとがおおいです   | 东京人多         |
| コーヒーは飲みません         | コーヒーはのみません           | 咖啡（的话）不喝 |

### 注意

- は 作助词时读 **wa**，不读 ha
- は 提示的是"已知信息"，が 提示的是"新信息"（见 はvsが 专题）

---

## が — 主语强调 / 对象

### 功能

A. **强调主语**（谁/什么做了动作、发生了什么事）
B. **能力/好恶的对象**（会~/喜欢~/讨厌~）

```
N + が
```

### 例句

| 功能     | 例句                         | 读法                             | 翻译             |
| -------- | ---------------------------- | -------------------------------- | ---------------- |
| 强调主语 | 雨が降っています             | あめがふっています               | 在下雨           |
| 强调主语 | 先生が来ました               | せんせいがきました               | 老师来了         |
| 能力对象 | 日本語がわかります           | にほんごがわかります             | 懂日语           |
| 好恶对象 | 猫が好きです                 | ねこがすきです                   | 喜欢猫           |

### 注意

- が 提示的通常是"新信息"或"焦点"
- 描述自然现象、眼前发生的事，用 が（雨が、風が、花が 等）
- 可能态、〜たい、好き/嫌い、上手/下手 的对象用 **が**

---

## を — 动作对象 / 经过 / 离开

### 功能

A. **动作的直接对象**（他动词的宾语）
B. **经过的场所**（移动经过某地）
C. **离开的起点**（从某地出发/下来）

```
N + を
```

### 例句

| 功能       | 例句                         | 读法                             | 翻译             |
| ---------- | ---------------------------- | -------------------------------- | ---------------- |
| 动作对象   | ご飯を食べます               | ごはんをたべます                 | 吃饭             |
| 动作对象   | 日本語を勉強します           | にほんごをべんきょうします       | 学日语           |
| 经过场所   | 公園を散歩します             | こうえんをさんぽします           | 在公园散步       |
| 经过场所   | 橋を渡ります                 | はしをわたります                 | 过桥             |
| 离开起点   | バスを降ります               | バスをおります                   | 下公交车         |
| 离开起点   | 家を出ます                   | いえをでます                     | 出家门           |

### 注意

- を 作助词时读 **o**，不读 wo
- 经过/离开的 を 后面跟的是**移动动词**（歩く、渡る、降りる、出る 等）

---

## に — 方向/时间点/对象/存在场所

### 功能总览

に 是 N5 功能最多的助词，考得也最多：

| 功能       | 含义                     | 例子                      |
| ---------- | ------------------------ | ------------------------- |
| 时间点     | 在某个时间               | 3時**に**                 |
| 移动方向   | 去/来/回某处             | 学校**に**行く            |
| 存在场所   | 在某处（有/在）          | 机の上**に**本がある      |
| 动作对象   | 动作的接受者             | 友達**に**あげる          |
| 频率基准   | 每~几次                  | 1日**に**3回              |
| 目的       | 去/来做某事（ます形+に）  | 食べ**に**行く            |

### 例句

| 功能       | 例句                           | 读法                             | 翻译               |
| ---------- | ------------------------------ | -------------------------------- | ------------------ |
| 时间点     | 7時に起きます                  | しちじにおきます                 | 七点起床           |
| 移动方向   | 明日、東京に行きます           | あした、とうきょうにいきます     | 明天去东京         |
| 存在场所   | 机の上に本があります           | つくえのうえにほんがあります     | 桌上有本书         |
| 动作对象   | 母に手紙を書きます             | ははにてがみをかきます           | 给妈妈写信         |
| 频率基准   | 1週間に2回、日本語を勉強します | いっしゅうかんににかい〜         | 每周学两次日语     |
| 目的       | 映画を見に行きます             | えいがをみにいきます             | 去看电影           |

### 注意

- 相对时间不加 に（きょう、あした、毎日 等），详见日期类考点
- 存在场所用 に（ある/いる），动作场所用 で（见下方的にvsで专题）

---

## で — 动作场所/手段/范围

### 功能总览

| 功能       | 含义                   | 例子                      |
| ---------- | ---------------------- | ------------------------- |
| 动作场所   | 在某地做某事           | 図書館**で**勉強する      |
| 手段/工具  | 用~（交通工具/语言/工具）| ペン**で**書く           |
| 范围限定   | 在~范围内（最高级）    | 日本**で**一番高い山      |
| 原因       | 因为~（客观原因）      | 台風**で**学校が休み      |
| 数量限定   | 几个人/几个（集合人数） | 3人**で**食事する         |
| 材料       | 用~材料制成            | 木**で**作る              |

### 例句

| 功能       | 例句                         | 读法                           | 翻译                 |
| ---------- | ---------------------------- | ------------------------------ | -------------------- |
| 动作场所   | 図書館で勉強します           | としょかんでべんきょうします   | 在图书馆学习         |
| 手段       | バスで学校へ行きます         | バスでがっこうへいきます       | 坐公交去学校         |
| 手段       | 日本語で話してください       | にほんごではなしてください     | 请用日语说           |
| 范围限定   | クラスで一番背が高いです     | クラスでいちばんせがたかいです | 班级里个子最高       |
| 原因       | 雨で試合が中止になりました   | あめでしあいがちゅうしになりました | 因雨比赛取消了   |
| 数量限定   | 一人で旅行します             | ひとりでりょこうします         | 一个人旅行           |
| 材料       | 紙で鶴を作ります             | かみでつるをつくります         | 用纸折鹤             |

### 注意

- で 强调的是"在哪里**做**什么"（动作），和に的"在哪里**有**什么"（存在）为必考对比（见下）

## に vs で — 存在场所 vs 动作场所（必考陷阱）

这是 N5 语法题中**最高频的对比考点**。

### 核心区分

```
に = 存在的场所（静态的"在哪儿"）
で = 动作的场所（动态的"在哪儿做什么"）
```

| 助词 | 功能     | 典型动词           | 例句                         |
| ---- | -------- | ------------------ | ---------------------------- |
| に   | 存在场所 | ある、いる、住む   | 東京**に**住んでいます       |
| で   | 动作场所 | 勉強する、食べる、働く | 東京**で**働いています   |

### 关键对比

```
公園に ベンチが あります。（公园里有长椅）    ← 存在：に
公園で サッカーを します。（在公园踢足球）    ← 动作：で
```

```
東京に 友達が います。（东京有朋友在）         ← 存在：に
東京で 友達に 会いました。（在东京见了朋友）   ← 动作：で
```

### 特殊：有的动词既能用に也能用で

「住む」虽然表示"居住"，但属于持续性状态，既可用 に（强调所在）也可用 で（强调生活行为）：

```
東京に住んでいます。（住在东京）— 标准的"存在/所在"
東京で住んでいます。（在东京生活）— 侧重"生活行为"
```

考试首选 **に**，更常见且更标准。

### 速记

> **在哪儿"有"用に、在哪儿"做"用で**

---

## へ — 移动方向

### 功能

指示**移动的大方向**，强调方向本身而非到达点。

```
N（地点）+ へ
```

### 例句

| 例句                         | 读法                           | 翻译             |
| ---------------------------- | ------------------------------ | ---------------- |
| 東京へ行きます               | とうきょうへいきます           | 去东京           |
| 家へ帰ります                 | うちへかえります               | 回家             |
| こちらの出口へどうぞ         | こちらのでぐちへどうぞ         | 请往这边出口走   |

### 注意

- へ 作助词时读 **e**，不读 he
- へ 可以和 に 互换的情况多，但也有区别（见下）

---

## へ vs に — 方向 vs 到达点（常考对比）

### 核心区分

```
へ = 强调"方向"（往哪儿去）
に = 强调"到达点/目的地"（去到哪儿）
```

| 例句                     | 比较                             |
| ------------------------ | -------------------------------- |
| 東京へ行く               | 往东京方向去（方向感强）         |
| 東京に行く               | 去东京（到了就到了，到此为止）   |

### 可以互换的情况

日常移动目的地，两者基本可互换：

```
学校へ行きます ✓
学校に行きます ✓
（都成立，へ偏方向感，に偏结果感）
```

### 不能互换的情况

に 的"存在场所"、"动作对象"、"时间点"功能，**不可用へ替换**：

```
机の上に本がある ✓       机の上へ本がある ✗
友達にあげる ✓           友達へあげる ✗
3時に会う ✓              3時へ会う ✗
```

### 速记

> **へ 是路上的箭头、に 是地图上画圈的目的地**

---

## は vs が — 主题 vs 主语（N5 最高难度）

这是整个 N5 助词体系中最难理解的对比，但考试中主要以固定句式区分。

### N5 阶段只需掌握的规则

**规则 1：大主语用 は、小主语用 が**

```
象は 鼻が 長いです。
ぞうは はなが ながいです。
大象鼻子长。（大象=大主题/は、鼻子=小主语/が）
```

**规则 2：疑问词作主语用 が，回答也对应 が**

```
Q: どれが あなたの かばんですか。
A: これが わたしの かばんです。
```

**规则 3：描述眼前看到的事实用 が**

```
あっ、バスが 来ました！
飛機が 飛んでいます。
```

**规则 4：固定句式中的 が**

| 句式                       | 对象助词 |
| -------------------------- | -------- |
| 〜が 好き/嫌い です        | が       |
| 〜が 上手/下手 です        | が       |
| 〜が わかります            | が       |
| 〜が あります/います       | が       |
| 〜が できます              | が       |
| 〜が ほしいです            | が       |
| 〜たい 的形式             | が/を 均可 |

**规则 5：は 管到句末、が 只管前面的谓语**

```
私は昨日買った本は面白かったです。
（两个は各自管不同的范围）
```

### 考试实战技巧

做选择题时问自己：

```
1. 是不是固定句式？（好き/上手/わかる → が）
2. 是不是眼前发生的事？（雨が/バスが → が）
3. 是不是疑问词？（だれが/どれが → が）
4. 以上都不是 → 大概率是 は（提示主题）
```

### 速记

> **新信息出场用が、已知主题用は；眼前事实用が、一般陈述用は**

## と — 和~/引用/条件

### 功能

A. **并列**（和~一起）
B. **互动对象**（和~做某事，必须有对方参与）
C. **引用**（说/想/写的内容）

### 例句

| 功能     | 例句                               | 读法                               | 翻译               |
| -------- | ---------------------------------- | ---------------------------------- | ------------------ |
| 并列     | パンと牛乳を買いました             | パンとぎゅうにゅうをかいました     | 买了面包和牛奶     |
| 互动对象 | 友達と遊びます                     | ともだちとあそびます               | 和朋友玩           |
| 互动对象 | 先生と話しました                   | せんせいとはなしました             | 和老师谈了话       |
| 引用     | 「ありがとう」と言いました         | ありがとうといいました             | 说了"谢谢"        |
| 引用     | 来週、試験があると思います         | らいしゅう、しけんがあるとおもいます | 我觉得下周有考试 |

### 注意

- 并列助词 と 是**全部列举**（把所有项都列出来），不完全列举用 や

### 并列助词对比：と vs や vs か

N5 常把这三个放在一起考，关键区别在于列举方式：

| 助词     | 列举方式       | 含义               | 例子                         |
| -------- | -------------- | ------------------ | ---------------------------- |
| と       | 全部列举       | A 和 B（就这些）   | パン**と**牛乳（就只有这两样） |
| や       | 不完全列举     | A 和 B 等等        | パン**や**牛乳（还有其他的）  |
| か       | 选择           | A 或 B             | パン**か**牛乳（选一个）     |

```
と = 限定（只有提到的）
や = 开放（还有没提到的）
か = 选择（二选一或多选一）
```

> N5 常见陷阱：因为と是"全部列举"，所以一般不用于"还有别人也做了"的语境中。例：✗ 私と行きました（如果你不是唯一一起去的人，这种"限定感"就不自然）。

## から — 起点/来源

日期类中已介绍过 から〜まで，这里补充其他用法。

### 功能

A. **时间/空间起点**
B. **来源**（从某人那里得到/听到）

### 例句

| 功能       | 例句                           | 读法                           | 翻译               |
| ---------- | ------------------------------ | ------------------------------ | ------------------ |
| 时间起点   | 9時から会議が始まります       | くじからかいぎがはじまります   | 九点开始会议       |
| 空间起点   | 駅から歩いて10分です           | えきからあるいてじゅっぷんです | 从车站走路十分钟   |
| 来源       | 友達から手紙が来ました         | ともだちからてがみがきました   | 朋友来信了         |
| 来源       | 先生から聞きました             | せんせいからききました         | 从老师那儿听说的   |
| 原材料     | 米から酒を作ります             | こめからさけをつくります       | 用米酿酒           |

### から vs を（离开）

- を = 离开的具体起点（家を出る）
- から = 起点/来源延续到某处（家から駅まで歩く）

---

## まで — 终点

### 功能

时间/空间的**终点**。

### 例句

| 例句                           | 读法                           | 翻译                 |
| ------------------------------ | ------------------------------ | -------------------- |
| 5時まで勉強します             | ごじまでべんきょうします       | 学习到五点           |
| 駅まで歩きます                 | えきまであるきます             | 走到车站             |
| 明日までに提出してください     | あしたまでにていしゅつしてください | 请在明天之前提交 |

### まで vs までに

- **まで** + 持续性动作（勉強する、待つ 等）：动作持续到该时间点
- **までに** + 一次性动作（提出する、返す 等）：在该时间点之前完成

```
5時まで待ちます。（等到五点 — 一直等）
5時までに帰ります。（五点前回来 — 一次性动作）
```

## も — 也

### 功能

表示"也"、"也（不）"。替换 は/が/を，加在其他助词后面。

### 例句

| 例句                         | 读法                         | 翻译             |
| ---------------------------- | ---------------------------- | ---------------- |
| 私も行きます                 | わたしもいきます             | 我也去           |
| 日本語もできます             | にほんごもできます           | 也会日语         |
| 何も食べませんでした         | なにもたべませんでした       | 什么也没吃       |

### も 的叠加规则

```
私は → 私も（替换は）
東京へ → 東京へも（加在へ后）
友達と → 友達とも（加在と后）
```

### 疑问词 + も — 全面肯定 / 全面否定

も 和疑问词结合后，表示"无论是谁/什么/哪儿/什么时候都~"。N5 选择题经常考这个搭配。

#### 否定句：疑问词 + も + ない — "什么也不~/谁也不~"

| 搭配           | 含义               | 例句                               | 翻译               |
| -------------- | ------------------ | ---------------------------------- | ------------------ |
| 何も〜ない     | 什么也不~          | 何も食べませんでした               | 什么也没吃         |
| 誰も〜ない     | 谁也不~            | 誰も来ませんでした                 | 谁也没来           |
| どこも〜ない   | 哪儿也不~          | どこも行きません                   | 哪儿也不去         |

> 「いつ」和も不组合为否定——「いつも」已是频率副词"总是"（与「たいてい/よく」同类）。"任何时候也"用いつでも。

#### 肯定句：疑问词 + でも — "什么都~/谁都~"

| 搭配           | 含义               | 例句                               | 翻译               |
| -------------- | ------------------ | ---------------------------------- | ------------------ |
| 何でも         | 什么都~            | 何でも食べます                     | 什么都吃           |
| 誰でも         | 谁都~              | 誰でもできます                     | 谁都会             |
| どこでも       | 哪儿都~            | どこでも寝られます                 | 哪儿都能睡         |
| いつでも       | 什么时候都~        | いつでも来てください               | 什么时候都可以来   |

> 否定用 **も**（何もない）、肯定用 **でも**（何でもある）。这是 N5 高频考点。

## の — 所属/内容说明

### 功能

A. **所属/所有**（的）
B. **内容说明**（同位语，解释前面的名词）
C. **名词化**（代替名词，避免重复）

### 例句

| 功能       | 例句                         | 读法                           | 翻译               |
| ---------- | ---------------------------- | ------------------------------ | ------------------ |
| 所属       | 私の本です                   | わたしのほんです               | 我的书             |
| 所属       | 田中さんのかばん             | たなかさんのかばん             | 田中先生的包       |
| 内容说明   | 日本語の勉強                 | にほんごのべんきょう           | 日语学习           |
| 内容说明   | 友達の山田さん               | ともだちのやまださん           | 朋友山田           |
| 名词化     | 赤いのが好きです             | あかいのがすきです             | 喜欢红色的（那个） |

## 助词叠加规则速查

| 叠法         | 含义         | 例子                   |
| ------------ | ------------ | ---------------------- |
