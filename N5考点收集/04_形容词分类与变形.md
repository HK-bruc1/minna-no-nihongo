# 形容词分类与变形

## 形容词两类

日语形容词分两类，区分关键在于词尾：

| 类别     | 日文名         | 特征                     | 例子                   |
| -------- | -------------- | ------------------------ | ---------------------- |
| い形容詞 | けいようし     | 基本形以「い」结尾       | 高**い**、安**い**     |
| な形容詞 | けいようどうし | 基本形不以「い」结尾，接名词需要「な」 | 静か**だ**、元気**だ** |

```
い形容詞 = 自带「い」尾巴，变形时改变这个尾巴
な形容詞 = 借「だ/です」结尾，变形时改变「だ/です」
```

---

## 变形体系概览

| 变形       | い形容詞例（高い） | な形容詞例（静かだ） | 用途                     |
| ---------- | ------------------ | -------------------- | ------------------------ |
| 现在肯定   | 高い               | 静かだ               | 基本形/简体现在时        |
| 现在否定   | 高くない           | 静かではない         | 否定                     |
| 过去肯定   | 高かった           | 静かだった           | 过去                     |
| 过去否定   | 高くなかった       | 静かではなかった     | 过去否定                 |
| て形       | 高くて             | 静かで               | 连接/并列                |
| 连名词     | 高い + N           | 静かな + N           | 修饰名词                 |
| 副词化     | 高く + V           | 静かに + V           | 修饰动词                 |

## い形容詞 全套变形

### 变化规则

```
い形容詞的变形 = 把词尾「い」替换为各种后缀

高い（たかい）
  ├── 现在肯定：たかい
  ├── 现在否定：たか  + くない  → たかくない
  ├── 过去肯定：たか  + かった  → たかかった
  ├── 过去否定：たか  + くなかった → たかくなかった
  ├── て形　　：たか  + くて    → たかくて
  ├── 连名词　：たかい + N       → たかい本
  └── 副词化　：たか  + く + V   → たかくとぶ
```

### 变形总表

| 变形       | 规则           | 高い →       | 安い →       | 楽しい →         |
| ---------- | -------------- | ------------ | ------------ | ---------------- |
| 现在肯定   | 〜い（原形）   | 高い         | 安い         | 楽しい           |
| 现在否定   | 〜くない       | 高**くない**   | 安**くない**   | 楽し**くない**     |
| 过去肯定   | 〜かった       | 高**かった**   | 安**かった**   | 楽し**かった**     |
| 过去否定   | 〜くなかった   | 高**くなかった** | 安**くなかった** | 楽し**くなかった** |
| て形       | 〜くて         | 高**くて**     | 安**くて**     | 楽し**くて**       |
| 连名词     | 〜い + N       | 高**い**本     | 安**い**本     | 楽し**い**本       |
| 副词化     | 〜く + V       | 高**く**飛ぶ   | 安**く**買う   | 楽し**く**遊ぶ     |

### 关键：い → く 变换

```
い形容詞所有变形（除了连名词）都需要先把「い」变成「く」

高い ──→ たか + く ──→ くない / かった / くて ...
         ↑
      这是核心动作
```

## な形容詞 全套变形

### 变化规则

```
な形容詞的变形 = 把词干后面的「だ」替换为各种后缀

静かだ（しずかだ）
  ├── 现在肯定：しずかだ / しずかです
  ├── 现在否定：しずか + ではない  → しずかではない
  ├── 过去肯定：しずか + だった    → しずかだった
  ├── 过去否定：しずか + ではなかった → しずかではなかった
  ├── て形　　：しずか + で        → しずかで
  ├── 连名词　：しずか + な + N    → しずかな町
  └── 副词化　：しずか + に + V    → しずかに話す
```

### 变形总表

| 变形       | 规则               | 静かだ →               | 元気だ →               | 好きだ →             |
| ---------- | ------------------ | ---------------------- | ---------------------- | -------------------- |
| 现在肯定   | 〜だ（原形）       | 静か**だ**             | 元気**だ**             | 好き**だ**           |
| 现在否定   | 〜ではない         | 静か**ではない**       | 元気**ではない**       | 好き**ではない**     |
| 过去肯定   | 〜だった           | 静か**だった**         | 元気**だった**         | 好き**だった**       |
| 过去否定   | 〜ではなかった     | 静か**ではなかった**   | 元気**ではなかった**   | 好き**ではなかった** |
| て形       | 〜で               | 静か**で**             | 元気**で**             | 好き**で**           |
| 连名词     | 〜な + N           | 静か**な**町           | 元気**な**人           | 好き**な**食べ物     |
| 副词化     | 〜に + V           | 静か**に**話す         | 元気**に**遊ぶ         | —                    |

### 礼貌体 vs 简体

な形容詞在口语/简体中用「だ」，礼貌体用「です」：

| 变形       | 简体                 | 礼貌体                   |
| ---------- | -------------------- | ------------------------ |
| 现在肯定   | 静かだ               | 静かです                 |
| 现在否定   | 静かではない         | 静かではありません       |
| 过去肯定   | 静かだった           | 静かでした               |
| 过去否定   | 静かではなかった     | 静かではありませんでした |

> N5 考试中礼貌体更常见，但题干中可能混入简体干扰。

## 形容词て形

て形用于连接多个形容词（"既~又~"）。

### 变化规则

| 类型       | 规则               | 例                       |
| ---------- | ------------------ | ------------------------ |
| い形容詞   | 去い + **くて**    | 高い → 高**くて**        |
| な形容詞   | 词干 + **で**      | 静かだ → 静か**で**      |

### 例句

| 例句                                         | 读法                                     | 翻译               |
| -------------------------------------------- | ---------------------------------------- | ------------------ |
| この部屋は 広くて 明るいです                 | このへやは ひろくて あかるいです         | 这个房间既宽敞又明亮 |
| あの店は 安くて おいしいです                 | あのみせは やすくて おいしいです         | 那家店又便宜又好吃 |
| 田中さんは 親切で 優しいです                 | たなかさんは しんせつで やさしいです     | 田中先生又热心又温柔 |
| この町は 静かで 便利です                     | このまちは しずかで べんりです           | 这个城镇又安静又方便 |

```
い形容詞て形 + い形容詞 → 広くて 明るい
い形容詞て形 + な形容詞 → 安くて 便利だ
な形容詞て形 + い形容詞 → 静かで 広い
な形容詞て形 + な形容詞 → 親切で 元気だ
```

> て形可以跨类连接，不受形容词类型限制。

## 形容词修饰名词

### い形容詞：直接接名词

```
い形容詞（原形）+ 名詞
```

| 例句                     | 读法                       | 翻译         |
| ------------------------ | -------------------------- | ------------ |
| 高い 本                  | たかい ほん               | 贵的书       |
| おいしい 料理            | おいしい りょうり         | 好吃的菜     |
| 古い 建物                | ふるい たてもの           | 旧的建筑     |

### な形容詞：加「な」再接名词

```
な形容詞（词干）+ な + 名詞
```

| 例句                     | 读法                       | 翻译         |
| ------------------------ | -------------------------- | ------------ |
| 静か**な** 町            | しずかな まち             | 安静的城镇   |
| 元気**な** 子供          | げんきな こども           | 精神的孩子   |
| 有名**な** 歌手          | ゆうめいな かしゅ         | 有名的歌手   |

> **这是 N5 最高频的陷阱之一。** な形容詞接名词时必须加「な」，不能直接用词干。

## 形容词修饰动词（副词化）

### い形容詞：い → く + V

| 例句                         | 读法                           | 翻译           |
| ---------------------------- | ------------------------------ | -------------- |
| **早く** 起きます            | はやく おきます               | 早起           |
| **大きく** 書いてください    | おおきく かいてください       | 请写大一点     |
| **安く** 買いました          | やすく かいました             | 便宜地买到了   |

### な形容詞：词干 + に + V

| 例句                         | 读法                           | 翻译           |
| ---------------------------- | ------------------------------ | -------------- |
| **静かに** 話してください    | しずかに はなしてください     | 请安静地说     |
| **上手に** 料理を作ります    | じょうずに りょうりをつくります | 擅长做菜      |
| **元気に** 遊んでいます      | げんきに あそんでいます       | 精神地玩着     |

```
い形容詞 副词化：い → く   （早い → 早く）
な形容詞 副词化：词干 + に （静かだ → 静かに）
```

## 特殊变形：いい/よい

形容词「いい」是 N5 唯一的**不规则い形容詞**。所有变形都基于古形「よい」：

| 变形       | いい →             | 规则                       |
| ---------- | ------------------ | -------------------------- |
| 现在肯定   | いい               | 只用「いい」               |
| 现在否定   | **よくない**       | 基于 よい，不是 ✗ いくない |
| 过去肯定   | **よかった**       | 基于 よい，不是 ✗ いかった |
| 过去否定   | **よくなかった**   | 基于 よい                  |
| て形       | **よくて**         | 基于 よい                  |
| 连名词     | いい + N           | いい本 ✓                   |

> 「いい」只在现在肯定时使用。「よくない」「よかった」等全部变形来自「よい」。N5 必考。

## 经典陷阱：容易误判为 い形容詞 的 な形容詞

有些单词看起来像い形容詞，实际上却是**な形容詞**，变形方式完全不同：

| 单词     | 类别     | 含义     | ✗ 错误变形（当い形容詞） | ✓ 正确变形（な形容詞） |
| -------- | -------- | -------- | ------------------------ | ---------------------- |
| きれい   | な形容詞 | 漂亮     | ✗ きれくない             | ✓ きれいではない       |
| 有名     | な形容詞 | 有名     | ✗ 有名くない             | ✓ 有名ではない         |
| 嫌い     | な形容詞 | 讨厌     | ✗ 嫌くない               | ✓ 嫌いではない         |
| 親切     | な形容詞 | 热心     | ✗ 親切くない             | ✓ 親切ではない         |
| 便利     | な形容詞 | 方便     | ✗ 便利くない             | ✓ 便利ではない         |

> **区分方法**：这些词的「い」不是词尾，而是汉字读音的一部分（きれ**い**、ゆうめ**い**）。看写法就能判断——写出来不是单独一个假名「い」在词尾。

```
きれい ── 最后的「い」是「綺麗」这个汉字词发音的一部分
高い   ── 最后的「い」是词尾，い形容詞的标志

汉字词 + い = 大概率是 な形容詞（きれい、有名、親切）
和语词 + い = 大概率是 い形容詞（高い、安い、早い）

⚠️ 例外：**嫌い**（きらい）是和语词但属于 な形容詞，需死记。
```

## い形容詞 vs な形容詞 关键对比

| 对比项       | い形容詞                     | な形容詞                       |
| ------------ | ---------------------------- | ------------------------------ |
| 基本形       | 〜い                         | 〜だ                           |
| 现在否定     | 〜**く** + ない              | 〜**ではない**                 |
| 过去肯定     | 〜**かっ** + た              | 〜**だっ** + た                |
| 过去否定     | 〜**くなかっ** + た          | 〜**ではなかった**             |
| て形         | 〜**くて**                   | 〜**で**                       |
| 连名词       | 直接 + N（**高い**本）       | **な** + N（**静かな**本）     |
| 副词化       | **く** + V（**早く**起きる）  | **に** + V（**静かに**話す）    |

### 速记对比

```
い形容詞 → 变形改「い」尾巴：くない / かった / くて
な形容詞 → 变形改「だ」尾巴：ではない / だった / で

い接名词不用な、な接名词必须な
い变副词用く、な变副词用に
```

## 经典考题陷阱

### 1. い形容詞否定形选择题

```
Q: この料理は（＿＿）です。
   ① おいしいくない    ② おいしくない
   ③ おいしじゃない    ④ おいしくて

おいしい → 去い + くない → おいしくない → ② ✓
① 多了「い」、③ 是 な形容詞 的否定方式
```

### 2. な形容詞接名词选「な」

```
Q: ここは（＿＿）公園です。
   ① 静か          ② 静かな
   ③ 静かだ        ④ 静かに

な形容詞 接名词必须加「な」→ ② ✓
① 缺少 な、③ 是终止形不能直接接名词、④ 是副词形
```

### 3a. いい 的否定变形

```
Q: きのうの映画はあまり（＿＿）。
   ① いかった      ② いいかった
   ③ よかった      ④ よくなかった

あまり + 否定 → よくない的过去否定 → よくなかった → ④ ✓
```

### 3b. いい 的过去肯定变形

```
Q: きのうの映画は（＿＿）。
   ① いかった      ② いいかった
   ③ よかった      ④ よくなかった

过去肯定 → よい的过去肯定 → よかった → ③ ✓
```

> 「いい」的全部变形用「よい」作基础。✗ いかった、✗ いくない 不存在。

### 4. い形容詞 vs な形容詞分类判断

```
Q: あの先生はとても（＿＿）。
   ① きれいです      ② きれくないです
   ③ きれかったです  ④ きれくてです

きれい是な形容詞 → ① ✓（きれいです）
②③④ 是当成了い形容詞的错误变形，全部不成立
```

### 5. て形连接跨类型

```
Q: あの人は（＿＿）、親切です。
   ① 若い           ② 若くて
   ③ 若かった       ④ 若いくて

い形容詞て形 + な形容詞 → 若くて + 親切 → ② ✓
```

## 速记口诀

```
い形容詞：去い变形、くない・かった・くて
な形容詞：去だ变形、ではない・だった・で
いい特殊：不好よくない、过去よかった
い接名词直接用、な接名词要加な
い变副词变成く、な变副词变成に
きれい有名不是い、别被假名骗了去
```
