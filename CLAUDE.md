# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

JLPT N5（日语能力考试N5级）备考资料库。存放考点整理、语法笔记和相关学习资源。

## Directory Structure

- `N5考点收集/` — N5 exam point notes organized by topic (e.g., dates/days of the week)
- `N5语法/` — N5 grammar PDFs: textbook, sentence pattern exercises, and answer keys
- `考点信息公告.md` — JLPT exam venue/info links

## File Conventions

- All content written in Chinese and Japanese
- Notes use Markdown with tables and mermaid diagrams where helpful
- PDF files for exercise materials and answer keys
- Exam points are organized by topic category (日期类, etc.)

## Git Conventions

- Commit messages use Chinese
- Commit when adding or updating exam point notes for a topic
