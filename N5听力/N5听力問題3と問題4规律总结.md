# N5听力 問題3＆問題4 规律总结

> **覆盖范围**：2010年7月 ~ 2025年12月，共8套N5真题  
> **核心特点**：不是长对话，而是考查**日常表达**和**即时应答**  
> **本质**：問題3 = 场景→选合适的发言；問題4 = 提问→选合适的回答  
> **好消息**：规律极强，可以总结成固定模式背诵

---

## 一、問題3（発話表現 / 即時応答）

### 1.1 题型特征

| 项目 | 说明 |
|------|------|
| **形式** | 给出一个场景描述 + 3个选项（有图或无图） |
| **问法** | 「何と言いますか」（要说什么？） |
| **本质** | 特定场景下，选择最合适的表达 |
| **数量** | 每套5题 |

### 1.2 六大场景类型

| 场景类型 | 说明 | 真题示例 |
|---------|------|---------|
| **① 寒暄语场景** | 进门/出门/回家/告别等 | 失礼します、ただいま、いってきます |
| **② 祝贺/慰问场景** | 生日/结婚/生病等 | おめでとう、お大事に |
| **③ 购物/请求场景** | 买东西/借东西/请求帮助 | 〜をください、〜ましょうか |
| **④ 社交/邀请场景** | 邀请/提议/拒绝 | 〜ませんか、〜ましょう |
| **⑤ 询问/求助场景** | 问时间/问路/求助 | 今何時、どこですか |
| **⑥ 反应/表达场景** | 对某事做出反应 | すごい、いいね、わかりました |

### 1.3 核心规律：三选一排除法

問題3的3个选项通常是这样设计的：

| 选项 | 类型 | 处理方式 |
|------|------|---------|
| **选项A** | 场景完全错误（张冠李戴） | 直接排除 |
| **选项B** | 场景相关但方向/对象错误 | 排除 |
| **选项C** | 完全匹配场景 | ✅ 正确答案 |

**排除技巧**：
1. 先看场景中的**关键动作**（进入房间？买东西？祝贺？）
2. 看选项的**动作方向**（给谁？向谁说？）
3. 排除明显不匹配的

### 1.4 高频真题举例

**【寒暄语类】**
- 进入老师房间 → **失礼します**（打扰了）
- 回家进门 → **ただいま**（我回来了）
- 出门上学 → **いってきます**（我走了）
- 朋友要走了 → **じゃあ、また**（再见）

**【祝贺类】**
- 朋友生日 → **おめでとう**（恭喜/生日快乐）
- 朋友结婚 → **おめでとうございます**（恭喜）

**【购物/请求类】**
- 买车票 → **京都まで一枚ください**（请给我到京都的一张）
- 想借用铅笔 → **铅笔、贷しましょうか**（要不要借你？）

**【询问类】**
- 想知道时间 → **今何時？**（现在几点？）
- 不懂汉字读法 → **これは何と読みますか**（这个怎么读？）

## 二、問題4（即時応答 / 発話に対する応答）

### 2.1 题型特征

| 项目 | 说明 |
|------|------|
| **形式** | 听一个短问题/陈述 + 3个选项（无图） |
| **问法** | 对方说一句话，你选最合适的应答 |
| **本质** | 疑问词对应 + 时态匹配 + 逻辑合理 |
| **数量** | 每套6题 |

### 2.2 核心规律：疑问词-回答匹配表

这是問題4最重要的规律！**疑问词和回答类型必须严格对应**。

| 疑问词 | 问什么 | 回答必须包含 | 干扰项常见套路 |
|--------|--------|-------------|---------------|
| **いつ** | 什么时候 | 时间点（去年、3月、火曜日） | 回答时长（5時間、3ヶ月） |
| **何時** | 几点 | 具体时间（9時、3時半） | 回答时长（1時間） |
| **何分** | 几分钟 | 时间长度（10分、15分） | 回答金额（千円） |
| **どこ** | 在哪里 | 地点（駅、教室、あそこ） | 回答时间/方式 |
| **どこで** | 在哪里做 | 地点+动作（デパートで買いました） | 回答方式/时间 |
| **どこか** | 去了哪里吗 | 具体地点（山に登りました） | 回答交通工具 |
| **何で** | 用什么方式 | 交通工具（自転車で、バスで） | 回答时间/人数 |
| **誰** | 是谁 | 人物（私の妹、田中さん） | 回答位置/特征 |
| **誰の** | 谁的 | 所有者（私の、田中さんの） | 回答种类/数量 |
| **何の** | 什么类型的 | 内容种类（料理の本、日本語の） | 回答所有权 |
| **いくつ** | 几个 | 数量（三つ、五冊） | 回答地点/动作 |
| **いくら** | 多少钱 | 金额（500円、千円） | 回答时间/数量 |
| **どのぐらい** | 多久/多长 | 时长/程度（2時間、とても） | 回答时间点 |
| **何** | 什么 | 具体事物/动作（本、食べる） | 回答 unrelated 内容 |
| **何を〜ますか** | 做什么 | 动作内容（映画を見る） | 回答时间/地点 |

### 2.3 三大排除法

#### 排除法①：答非所问（最常用）

问题问A，选项回答B → 直接排除

| 问题 | 错误选项 | 为什么错 |
|------|---------|---------|
| 図書館はどこですか | 6時までです | 问地点，回答时间 |
| 何時に来ますか | バスで来ます | 问时间，回答方式 |
| 誰の本ですか | 五冊です | 问归属，回答数量 |
| 何の本ですか | 私の本です | 问类型，回答归属 |

#### 排除法②：时态错配

问题用过去时，选项用现在时/将来时 → 排除

| 问题 | 错误选项 | 为什么错 |
|------|---------|---------|
| 子供のとき、よく本を読みましたか | 毎週図書館に行きます | 过去时→现在时 |
| 夏休みはどこかへ出かけましたか | 旅行しましょう | 过去时→将来提议 |
| 先週の日曜日、どこかへ行きましたか | 海に行きましょう | 过去时→将来提议 |

> **关键信号**：选项中出现 **〜ましょう**（~吧）→ 一定是将来提议，如果问题问的是过去/现在 → 直接排除

#### 排除法③：前提否定

对方问题的前提不正确，需要修正前提

| 问题 | 正确回答 | 为什么 |
|------|---------|--------|
| どこで買いましたか | 私が作りました | 不是买的，是自己做的 |
| いつ日本に来ましたか | 去年です | 直接回答时间点 |

### 2.4 功能表达对应表

除了疑问词匹配，还有一些**功能型表达**需要对应：

| 对方说的话 | 功能 | 合适回应 | 不合适回应 | 真题出处 |
|-----------|------|---------|-----------|---------|
| 〜ましょうか（要我~吗？） | 主动帮助 | ありがとうございます | どういたしまして | 2018.7-3-3 |
| 〜ませんか（要不要~？） | 提议/邀请 | そうしましょう | 時期関連の答え | 2010.7-3-1 |
| もう〜ましたか（已经~了吗？） | 确认完成 | はい/いいえ、今からです | 〜ましょう | 2018.7-4-2 |
| 美味しい〜ですね（很好吃呢） | 赞赏+提问 | 私が作りました（否定前提） | はい、そうです（仅附和） | 2018.7-4-3 |
| 知っていますか（知道吗？） | 询问信息 | はい、わかりますよ | 電話をしていません | 2018.7-4-5 |
| いつ〜来ましたか（什么时候来的？） | 问时间点 | 去年です | 5時間です（时长） | 2018.7-4-1 |
| 〜てもいいですか（可以做~吗？） | 请求许可 | はい、どうぞ | いいえ、ありません | 常见 |

## 三、問題3＆問題4 共通的核心能力

### 3.1 必须掌握的寒暄语（10个）

| 寒暄语 | 使用场景 | 出现频率 |
|--------|---------|---------|
| **いってきます** | 离开家/公司出门时说 | ★★★★★ |
| **ただいま** | 回到家/公司时说 | ★★★★★ |
| **おかえりなさい** | 迎接回来的人时说 | ★★★★ |
| **いらっしゃいませ** | 店员对顾客说 | ★★★★ |
| **失礼します** | 进入他人房间/办公室时说 | ★★★★★ |
| **お邪魔します** | 去别人家拜访时说 | ★★★ |
| **じゃあ、また** | 和朋友告别时说 | ★★★★ |
| **おめでとう** | 祝贺生日/结婚/毕业等 | ★★★★ |
| **いただきます** | 吃饭前说 | ★★★ |
| **ごちそうさまでした** | 吃完饭后说 | ★★★ |

### 3.2 必须掌握的句型（8个）

| 句型 | 含义 | 使用场景 | 出现频率 |
|------|------|---------|---------|
| **〜をください** | 请给我~ | 购物/请求物品 | ★★★★★ |
| **〜てください** | 请做~ | 请求对方做某事 | ★★★★★ |
| **〜ましょうか** | 要不要我~？ | 主动提议帮助 | ★★★★ |
| **〜ませんか** | 要不要~？ | 提议/邀请 | ★★★★ |
| **〜ましょう** | ~吧（一起做） | 提议共同行动 | ★★★★ |
| **もう〜しましたか** | 已经~了吗？ | 确认完成 | ★★★★ |
| **〜ています** | 正在~/习惯性~ | 描述状态 | ★★★★ |
| **〜から** | 因为~ | 说明原因 | ★★★★ |

## 四、备考建议

### 4.1 2周速通计划

| 阶段 | 时长 | 内容 |
|------|------|------|
| **第1周** | 15分钟/天 | 背熟"寒暄语10个"+"句型8个"+"疑问词匹配表" |
| **第2周** | 20分钟/天 | 精做8套真题的問題3+問題4，每题分析排除法 |

### 4.2 三步解题法

```
Step 1: 听问题/看场景 → 抓准疑问词或场景关键词
         ↓
Step 2: 扫选项 → 用排除法去掉明显错误的
         ↓
Step 3: 确认答案 → 时态/疑问词/场景 三重匹配
```

### 4.3 错题分析模板

| 错题 | 错误原因 | 属于哪种规律 |
|------|---------|-------------|
| 疑问词没听清 | いつvs何時vs何分 | 疑问词匹配表 |
| 时态没注意 | ましたか→ましょう | 时态错配排除 |
| 场景搞混 | 失礼しますvsお邪魔します | 寒暄语场景 |
| 答非所问 | どこ→時間で答える | 答非所问排除 |

## 五、真题规律总结

### 5.1 問題3 高频场景（按出现频率排序）

1. **寒暄语选择**（进门/出门/回家/告别）
2. **购物请求**（〜をください）
3. **祝贺表达**（おめでとう）
4. **主动帮助**（〜ましょうか）
5. **询问表达**（今何時、どこですか）

### 5.2 問題4 高频疑问词（按出现频率排序）

1. **いつ**（什么时候）→ 回答时间点
2. **どこ/どこで/どこか**（哪里）→ 回答地点
3. **誰/誰の**（谁/谁的）→ 回答人物
4. **何時**（几点）→ 回答具体时间
5. **何で**（用什么方式）→ 回答交通工具
6. **もう〜ましたか**（已经~了吗？）→ 肯定/否定回答
7. **いくつ/いくら**（几个/多少钱）→ 回答数量/金额
8. **知っていますか**（知道吗？）→ はい/いいえ + 相关内容
9. **〜てもいいですか**（可以做~吗？）→ はい、どうぞ / いいえ

### 5.3 最常见的3种错误选项设计

| 套路 | 识别方法 | 出现频率 |
|------|---------|---------|
| **答非所问** | 问地点答时间、问时间答方式 | ★★★★★ |
| **时态错配** | 问题问过去，选项用〜ましょう | ★★★★★ |
| **张冠李戴** | 寒暄语用错场景 | ★★★★ |

## 六、核心口诀

> **問題3：看场景→选表达→排除张冠李戴**
> 
> **問題4：听疑问词→匹配回答类型→排除答非所问和时态错配**
> 
> **万能排除法：答非所问直接删，时态错配直接删，场景不对直接删**
