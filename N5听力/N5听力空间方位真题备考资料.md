## 一、题型概述

### 1.1 题型定位

| 项目 | 内容 |
|------|------|
| 所属题型 | 課題理解（もんだい1） |
| 出现位置 | 第 1 大题（共约 7 题）的前半部分，通常第 1~4 题 |
| 出现频率 | **每回考试约 2~3 题**，是课题理解的高频核心考点 |
| 选项形式 | **4 选 1**（配有图片/地图，需选择正确的图示） |
| 音频播放 | 仅播放一次，不可暂停或重听 |

### 1.2 考什么？

这类题目考察的核心能力是：
1. **听懂日语中描述空间位置关系的表达**（前、后、左、右、隔壁、对面等）
2. **结合图示判断正确方位**（地图题、室内布局题、物品摆放题等）
3. **抓住对话中的关键转折与修正**（常出现「先错后对」的陷阱）

## 二、出题形式全总结

### 2.1 提问方式（5 种句式）

不要只以为问法就是「在哪里」。真题中出现过以下 **5 种提问句式**：

| 序号 | 提问句式 | 示例 | 出现频率 |
|------|----------|------|----------|
| ① | **～はどこですか** | 男の人はどこに行きますか | 极高频 |
| ② | **～はどちらですか** | 郵便局はどちらですか | 高频 |
| ③ | **～はどこにありますか** | トイレはどこにありますか | 中频 |
| ④ | **～はどっちですか** | 駅はどっちですか | 中频（口语式） |
| ⑤ | **～はどの～ですか** | どの部屋に入りますか | 低频 |

> **老师提示**：
> - 「どこ」是「哪里」的基本说法
> - 「どちら」是更礼貌的说法，在听力中**非常常见**
> - 「どっち」是口语说法，虽然 N5 中出现不多，但要能听懂
> - 问题开头一定会说明问的是「男の人」还是「女の人」，**务必听清**！

### 2.2 对话模式

真题中位置题的对话模式主要有 3 种：

| 模式 | 示例 | 说明 |
|------|------|------|
| **问路型** | 男の人が道を聞いている | 问某个场所（邮局、银行等）怎么走 |
| **物品放置型** | 荷物をどこに置きますか | 讨论物品放在哪里（桌上、冰箱旁等） |
| **目的地确认型** | この後どこへ行きますか | 确认某人接下来要去哪里 |

### 真题 1：2012年12月 第1大题 第6题（25年7月复现题）

#### 题目信息
| 项目 | 内容 |
|------|------|
| 考试场次 | 2012年12月 / 2025年7月（几乎原题复现！） |
| 题型 | 问路型（街道布局图） |
| 核心方位词 | 左に、隣が、間です |

#### 图示类型

```
街道布局图：

[本屋] --- [レストラン] --- [郵便局] --- [銀行]
            （← 4个选项的位置排列不同，需选出正确的相对位置）
```

> 图示为一条街道上排列着多家店铺/建筑，四幅图的区别在于各建筑物的**相对位置关系**不同。

#### 听力原文

```
男の人と女の人が話しています。男の人はどこに行きますか。

男：すみません。郵便局に行きたいですが。どこですか。
女：ああ、郵便局ですね。あそこに本屋がありますね。
男：はい。
女：本屋の左にレストランがあります。
男：はい。
女：レストランの隣が郵便局です。レストランと銀行の間です。
男：分かりました。ありがとうございます。
```

#### 参考译文

```
男人：不好意思，我想去邮局，请问在哪里？
女人：啊，邮局啊。那里有家书店对吧。
男人：是的。
女人：书店的左边有一家餐厅。
男人：是的。
女人：餐厅的隔壁就是邮局。在餐厅和银行之间。
男人：明白了，谢谢您。
```

#### 逐句解析

| 句子 | 考点 | 方位推理 |
|------|------|----------|
| 本屋の**左**にレストランがあります | 「左に」= 在左边 | 书店 ← 餐厅 |
| レストランの**隣**が郵便局です | 「隣」= 隔壁 | 餐厅 → 邮局 |
| レストランと銀行の**間**です | 「間」= 之间 | 餐厅—邮局—银行（邮局在中间） |

#### 正确位置关系

```
本屋 → [左] → レストラン → [隣/間] → 郵便局 → 銀行
                    ↑
              （邮局在餐厅和银行之间）
```

> **老师提示**：这道题出现了 **3 个方位词叠加描述**（左→隣→間），是位置题中的经典多层方位题。做这种题要在脑中画一条线，按顺序排列各建筑物。

### 真题 3：2021年12月 第1大题 第3题

#### 题目信息
| 项目 | 内容 |
|------|------|
| 考试场次 | 2021年12月 |
| 题型 | 物品放置型（室内场景图） |
| 核心方位词 | 上、横 |

#### 图示类型

```
室内场景图：
┌─────────────────┐
│    [テーブル]     │  ← 选项差异：行李放在哪里？
│        ↑         │     A. テーブルの上
│ [冷蔵庫] [椅子]   │     B. 冷蔵庫の上
│        ↑         │     C. 冷蔵庫の横の椅子
│    (荷物?)       │     D. テーブルの下
└─────────────────┘
```

> 这是一道**室内物品摆放题**，与街道地图题完全不同。四幅图的区别在于行李放在了不同的位置。

#### 听力原文

```
男の子は荷物をどこに置きますか

男：この荷物はどこに置きますか
女：テーブルの上に置いてください
男：でも、テーブルの上は雑誌がありますよ
女：そうですか、じゃあ、冷蔵庫の上はどうですか
男：上に置きますか、ちょっと高いですよね
女：じゃ、冷蔵庫の横の椅子に置いてください
男：はい、わかりました
```

#### 参考译文

```
男：这个行李要放在哪里？
女：请放在桌子上。
男：但是，桌子上有杂志啊。
女：是吗，那放在冰箱上面怎么样？
男：放在上面吗，有点高啊。
女：那就放在冰箱旁边的椅子上吧。
男：好的，明白了。
```

#### 解题关键——**转折链追踪**

这道题是典型的 **"否定链"题型**，经历了 **3 次转折**：

| 轮次 | 提议位置 | 结果 | 否定关键词 |
|------|----------|------|------------|
| 第1次 | テーブルの上 | ❌ 否定 | でも、雑誌がありますよ |
| 第2次 | 冷蔵庫の上 | ❌ 否定 | ちょっと高いですよね |
| 第3次 | 冷蔵庫の**横の椅子** | ✅ **正解** | じゃ～（最终方案） |

> **老师提示**：
> 1. 这道题用到了「上」和「横」两个垂直/水平方位词，是**室内场景题的典型模式**
> 2. 「じゃ」是解题的关键信号词——当对话中出现「じゃ（那/那么）」时，后面往往是**最终决定**
> 3. 要注意区分 **～の上（上面）** 和 **～の横（旁边）**，这是室内场景最常见的两个方位词

### 真题 4：2021年12月 第1大题 第4题

#### 题目信息
| 项目 | 内容 |
|------|------|
| 考试场次 | 2021年12月 |
| 题型 | 目的地确认型（街道布局图） |
| 核心方位词 | 隣、そば、前 |

#### 听力原文

```
女の人はこの後どこへ行きますか

男：この後、どこかへ行きますか
女：ええ、駅の隣の喫茶店へ行きます
男：ああ、駅のそばの喫茶店ですか、いいですね
女：いいえ、駅の隣ではなくて、大学の前の喫茶店ですよ
```

#### 参考译文

```
男：之后你要去哪里吗？
女：嗯，去车站旁边的咖啡店。
男：啊，是车站附近的咖啡店吗，不错啊。
女：不是，不是车站旁边的那家，是大学前面的那家咖啡店哦。
```

#### 解题关键——**否定修正型（核心陷阱！）**

这道题是 N5 方位题中**最典型的陷阱题型**：

| 阶段 | 内容 | 关键词 |
|------|------|--------|
| 第一次说 | 駅の**隣**の喫茶店 | 隣（旁边） |
| 男的确认 | 駅の**そば**の喫茶店 | そば（附近） |
| 女的否定 | **いいえ**、駅の隣では**なくて** | 否定！ |
| 最终答案 | **大学の前**の喫茶店 | 前（前面） |

> **老师提示**：
> 1. **「いいえ、～ではなくて」** 是位置题中最高频的否定+修正表达，听到这个结构一定要**把注意力集中在「なくて」后面**
> 2. 「隣」和「そば」意思接近但有区别：
>    - **隣（となり）** = 紧挨着（隔壁）
>    - **そば（そば）** = 附近（可稍远）
> 3. 这道题同时考查了 **否定修正** 和 **两个易混方位词的辨析**（隣 vs そば），是备考必练的经典题

## 五、图示类型分类与应对策略

### 5.1 五大图示类型

| 类型 | 子类型 | 说明 | 真题示例 |
|------|--------|------|----------|
| **街道/室外类** | 十字路口 | 多条道路交叉，标注店铺 | 25-7-1.6（郵便局问路） |
| | 一条街道 | 沿线排列多家店铺 | 12-12-1.6（郵便局问路） |
| | 街区地图 | 多个建筑物和道路平面图 | - |
| **室内布局类** | 房间布局 | 沙发、电视、书架等家具位置 | 21-12-1.3（荷物放置） |
| | 物品摆放 | 桌上物品的位置关系 | - |
| **平面位置图** | 相对位置图 | A、B、C 的相对方位 | 21-12-1.4（咖啡店位置） |
| **场景选择类** | 4幅场景图 | 从中选择对话描述的场景 | - |

### 5.2 各类图示的解题要点

**街道类（最常见）：**
- 先看 4 幅图的**差异点**——哪两个建筑物的位置关系不同
- 听对话时重点抓 **A の 左/右/隣/前/間 に B** 的结构
- 在草稿纸上按顺序画出建筑物排列

**室内类（次常见）：**
- 先看 4 幅图中物品放在了哪个位置（上/下/横/隣）
- 注意对话中的 **「じゃ」** ——表示最终方案
- 垂直方位（上/下）和水平方位（横/隣）容易混淆，要听清

**平面位置类：**
- 先看图中有哪几个地标
- 注意 **否定修正**（いいえ、～ではなくて）
- 听清最终确认的位置

## 六、解题技巧与常见陷阱

### 6.1 标准解题步骤（4步法）

```
步骤1：先看图（5秒）
    ↓ 找出4幅图的差异点（通常在2~3个方位上不同）
步骤2：听清问题
    ↓ 确认问的是「男の人」还是「女の人」
步骤3：抓方位词
    ↓ 听到方位词→立刻在脑中对应到图上→排除错误选项
步骤4：确认最终答案
    ↓ 注意「じゃ」「いいえ」后的修正才是正确答案
```

### 6.2 4大常见陷阱

| 陷阱名称 | 说明 | 应对方法 | 真题示例 |
|----------|------|----------|----------|
| **先错后对陷阱** | 先说一个位置，然后否定再给出正确位置 | 重点听「いいえ」「じゃ」「ちょっと」之后的内容 | 21-12-1.4（咖啡店） |
| **多层方位叠加** | 连续使用多个方位词描述，容易遗漏 | 按顺序画排列图 | 12-12-1.6 / 25-7-1.6（邮局） |
| **男女动作混淆** | 男女去不同地方或做不同动作 | 看清问题问的是谁 | - |
| **近义方位干扰** | 使用意思相近的方位词制造混淆 | 区分隣/そば/近く、上/横等 | 21-12-1.4（隣→そば→前） |

### 6.3 关键信号词

听到这些词，意味着**答案即将出现或刚被否定**：

| 信号词 | 含义 | 作用 |
|--------|------|------|
| **いいえ / いや** | 不是 | 否定前面的内容，正确答案在后面 |
| **～ではなくて** | 不是～而是 | 否定+修正，后面的是正确答案 |
| **じゃ / じゃあ** | 那么/那就 | 经过讨论后的最终决定 |
| **でも** | 但是 | 转折，后面通常是修正 |
| **ちょっと** | 有点～ | 委婉否定前面方案，后面会提出新的 |
| **やっぱり** | 还是 | 改变主意，回到最初方案或新方案 |

## 七、真题复现规律与备考建议

### 7.1 真题复现现象

> **重大发现**：JLPT N5 听力存在**原题或近乎原题复现**的现象！

| 原题 | 复现题 | 相似度 |
|------|--------|--------|
| 2012年12月 第1大题 第6题 | 2025年7月 第1大题 第6题 | **约 95%**（仅个别用词有微小差异） |

这意味着：**做历年真题是性价比最高的备考方式**。

### 7.2 核心备考清单

```
□ 1. 熟背「4.1 基本方位词」（上/下/前/後ろ/左/右/中/外）
□ 2. 熟背「4.2 相对方位词」（隣/横/そば/近く/向かい/間/手前/奥）
□ 3. 熟背「4.4 常见场所词」（駅/郵便局/銀行/喫茶店/本屋 等）
□ 4. 精练以上 5 道真题，做到每句都听懂、每个方位词都理解
□ 5. 练习「先看图→听对话→抓方位→排干扰」的解题流程
□ 6. 注意培养对「いいえ」「じゃ」「でも」等信号词的敏感度
```

### 7.4 考场实战建议

1. **发下试卷后立刻看第 1 大题的图片**——位置题的图片是最需要提前分析的
2. **画出方位关系**——如果允许打草稿，快速画出建筑物/物品的排列顺序
3. **不要纠结于每个词的意思**——位置题不需要听懂全文，**抓住方位词就能解题**
4. **听到否定信号立刻警觉**——「いいえ」「じゃ」后面往往是答案
5. **如果不确定，根据最后听到的方位选**——正确答案通常在对话末尾

## 八、自测练习

请根据以下真题原文，画出正确的位置关系图，并写出答案：

### 练习 1（2012年12月第6题 / 2025年7月第6题 同类型）

```
男の人はどこに行きますか。

男：すみません。郵便局に行きたいですが。どこですか。
女：あそこに本屋がありますね。本屋の左にレストランがあります。
男：はい。
女：レストランの隣が郵便局です。レストランと銀行の間です。
```

<details>
<summary>点击查看答案</summary>

**正确答案**：レストランと銀行の間にある郵便局

**位置关系**：
```
本屋 → レストラン → 郵便局 → 銀行
        ↑（左）     ↑（隣/間）
```

**关键方位词**：左に → 隣が → 間です（3个方位词连续叠加）
</details>

### 练习 2（2021年12月第3题 同类型）

```
男の子は荷物をどこに置きますか。

男：この荷物はどこに置きますか
女：テーブルの上に置いてください
男：でも、テーブルの上は雑誌がありますよ
女：そうですか、じゃあ、冷蔵庫の上はどうですか
男：上に置きますか、ちょっと高いですよね
女：じゃ、冷蔵庫の横の椅子に置いてください
```

<details>
<summary>点击查看答案</summary>

**正确答案**：冷蔵庫の横の椅子

**转折链**：
```
テーブルの上（× 有杂志）
  → 冷蔵庫の上（× 太高）
  → 冷蔵庫の横の椅子（✓ 最终决定）
```

**信号词**：でも → じゃあ → ちょっと → じゃ
</details>
